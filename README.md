# gridsemblefdr
This is a placeholder for the gridsemble R package introduced in *Ensemble Methods for False Discovery Rates*. This page will be updated as soon as the package is complete.

## Overview
gridsemblefdfr is a novel approach to tail end (Fdr) and local (fdr) false discovery rate estimation that builds upon existing methods in two important ways. First, we utilize ensemble methods to combine information from models with different hyperparameters. Simulation studies show that on average, an ensemble outperforms existing methods with default hyperparameters. Second, we determine informed sets of hyperparameters to ensemble over through a grid search on simulated, labeled test statistics based on the true, unlabeled data. Simulation studies show that in an ensemble, these informed sets of hyperparameters outperform random sets of hyperparameters.
